# How to run App
```
(MultiRoute) $ julia --project
julia> ]
pkg> instantiate
pkg> <ctrl + d> # quit julia

(MultiRoute) $ bin/server # Linux / Mac
(MultiRoute) $ bin/server.bat  # windows
```
